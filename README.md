# API Testing Project

This project is dedicated to testing API requests for a book collection application. It includes API endpoints for managing books, such as retrieving, creating, updating, and deleting records.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **GET**: Retrieve a list of books or filter them by language.
- **POST**: Add a new book to the collection.
- **DELETE**: Remove a book from the collection by ID.
- **PUT**: Update book details.

  
## Technologies Used

- **Node.js** with TypeScript
- **Next.js** for API routes
- **Jest** for testing
- **Mocking** with Jest for isolating tests
- **JSON** file for data storage
